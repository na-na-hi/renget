##(FAQ)Programs Used
**YT2MP3** - I use my own program to do this but you can use one of the countless websites to aquire your desired music as a .mp3 file 
**UVR5** - I downloaded and use this program locally on my pc to separate my .mp3 file into Vocals and Instrumental. There are several alternatives you can find and run virtually online, however there is a usually a queue but its not too bad.
**RVC v2** - I use a virtual version of this program to change the Vocals track into the AI cover variant of your desired song.
**Model** - You most importantly need to find your desired Model for your AI to use when generating the voice component. You can find these online or from a discord server, just make sure you find one that is compatible with RVC format if you decide to use that program. 
**Audacity** - I use this for mixing the final product, while it it quite simplistic and easy to use, it is a free alternative to something like FL Studio, which I have heard is a better program but I personally dont know how to use it correctly yet.
