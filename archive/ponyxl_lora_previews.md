#Previews for the LoRAs on ponyxl_loras_n_stuff

These all have the same set of prompts and seeds. Each example image has a slightly different prompt format  with the usage of score and source tags which is why they can look pretty different for the same LoRA, this was done because of how the LoRAs are inconsistently trained/tested. I also made no attempts to clean up any of the images so they can look a bit chuffed, it's not necessarily representative of how it will behave when you use with some good *prompt engineering* but hopefully the images can give you an idea of how the style will look at least. Each thumbnail is a link to the full image on catbox.
